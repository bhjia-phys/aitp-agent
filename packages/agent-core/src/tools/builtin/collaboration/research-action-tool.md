Inspect and record Hakimi research actions.

Use this tool to list available semantic research actions, render the primitive tool plan for an action, ask the scheduler which action should handle open obligations next, open/switch/list/close active WorkFrames, compile/load bounded ContextPacks for a WorkFrame, inspect the attached DomainPack manifest, list/load WorkFrame-scoped evidence refs, inspect AITP runtime payload profile metadata, inspect/search AITP curated heuristic RAG context, draft read-only curated RAG promotion plans and prefilled write-bridge call drafts, run deterministic in-process research executors, execute a configured AITP write bridge, or record the outcome of a semantic research action. This tool can execute only built-in graph queries, formalization-blueprint export, registered benchmark adapters, scoped ResearchLedger evidence inspection, read-only AITP runtime payload profile inspection, read-only AITP curated RAG inspection/search/draft planning, and explicitly configured AITP write-bridge operations. It does not execute arbitrary shell, git, web, MCP, or HPC work by itself. Primitive tool execution should remain attributed separately and then recorded back through the action result.
